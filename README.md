هذا المشروع يقدّم تطبيقًا مبسّطًا لهيكل بيانات الـ Circular Buffer  
مكوّنات الهيكل
يحتوي الـ CircularBuffer على:

buffer[] : مصفوفة ثابتة الحجم لتخزين البيانات

head : موقع القراءة

tail : موقع الكتابة

count : عدد العناصر الحالية داخل البافر
الدوال الأساسية
✔️ init(cb)
تهيئة المؤشرات والقيم الابتدائية.

✔️ isFull(cb)
تتحقق إذا كان البافر ممتلئًا:
isEmpty(cb)
تتحقق إذا كان البافر فارغًا:
write(cb, data)
تضيف عنصرًا جديدًا إلى البافر إذا لم يكن ممتلئًا.
إذا كان ممتلئًا → تظهر رسالة "Buffer Overflow".

✔️ read(cb)
تقرأ عنصرًا من البافر إذا لم يكن فارغًا.
إذا كان فارغًا → تظهر رسالة "Buffer Underflow".
وصف عمل البرنامج
يقرأ البرنامج اسمًا من المستخدم.

يضيف إليه اللاحقة "CE-ESY".

يكتب كل حرف من الاسم داخل الـ Circular Buffer.

يقرأ الحروف من البافر ويطبعها بنفس الترتيب.
